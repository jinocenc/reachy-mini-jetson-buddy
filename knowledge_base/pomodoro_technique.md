# Pomodoro Technique

The Pomodoro Technique is a time management method developed by Francesco Cirillo in the late 1980s.

## Standard Configuration

- Work period: 25 minutes (one "Pomodoro")
- Short break: 5 minutes
- Long break: 15 to 30 minutes after 4 Pomodoros
- One full cycle of 4 Pomodoros takes approximately 2 hours

## Rules

1. Decide on the task to be done
2. Set the timer for 25 minutes
3. Work on the task until the timer rings
4. Take a short 5 minute break
5. After four Pomodoros, take a longer 15 to 30 minute break
6. If a distraction pops into your head, write it down and return to the task
7. A Pomodoro is indivisible — if interrupted, the Pomodoro must be abandoned or the interruption must be postponed

## Common Modifications

- Shorter Pomodoros (15 or 20 minutes) for tasks requiring intense concentration
- Longer Pomodoros (45 or 50 minutes) for deep work and creative tasks
- Extended breaks (20 to 30 minutes) for physically demanding study sessions
- Flexible break activities: stretching, hydration, eye rest, short walks

## When to Suggest Modifications

- If the user consistently finishes early, suggest shorter Pomodoros
- If the user is frequently interrupted at the timer, suggest longer Pomodoros
- If the user seems fatigued, suggest longer breaks
- If the user is in a flow state near the end of a Pomodoro, allow a brief extension before break
